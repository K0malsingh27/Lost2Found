# Lost2Found - Lost Item Recovery System

**Lost2Found** is a simple web-based application that helps people post lost items and get help from others to find them. By scanning a unique QR code, users can share details about the lost item and get contacted by the person who finds it.

## Features
- Post lost items with details and contact information
- View and contact people who found your lost items
- Simple, user-friendly interface

## How to Run
1. Download or clone the repository.
2. Open the `index.html` file in any browser.
