# Lost2Found - Lost Item Recovery System

## 📌 Project Description

Lost2Found is a simple web-based application that helps users report lost items and allows others to help return them by contacting the owner.

---

## ⚙️ How It Works

1. User opens the website
2. Fills the form with:

   * Item name
   * Description
   * Location
   * Contact details
3. Clicks **Post Lost Item**
4. The post is displayed on the website
5. Finder contacts the owner

---

## 💡 Features

* Post lost items easily
* View all lost item posts
* Stores data using browser LocalStorage
* Works on both mobile and desktop

---

## 🛠️ Technologies Used
